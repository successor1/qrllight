# qrllight
QRL lightweight wallet based on PyQt5

# How to send to multiple recipients

QRL address format: address1 address2  
Example: Q01..fb8eeaaa3e570808063 Q01..fb8eeaaa3e570808063  
Amount format: amount1 amount2  
Example: 2.5 2.5
